# VoF.jl

Volume-of-Fluid two-phase flow extension for [WaterLily.jl](https://github.com/WaterLily-jl/WaterLily.jl).
Private, WIP. See [PLAN.md](./PLAN.md) for the full design.

Algebraic VoF with a MULES-style compressive limiter, continuum surface
force model for surface tension. Built on the WaterLily scalar-transport
hook (see PLAN 1).
